# Create Your GitHub Repository First

## The repository doesn't exist yet. Here's how to create it:

### Step 1: Go to GitHub
1. Open your browser
2. Go to [github.com](https://github.com)
3. Make sure you're logged in as `ohmyraul`

### Step 2: Create New Repository
1. Click the **"+"** icon in the top right corner
2. Click **"New repository"**

### Step 3: Fill in the Details
- **Repository name:** `wedding-website`
- **Description:** "Shubs & Alysha Wedding Website" (optional)
- **Visibility:** Choose **Public** or **Private** (your choice)
- **IMPORTANT:** Do NOT check any of these boxes:
  - ❌ Add a README file
  - ❌ Add .gitignore
  - ❌ Choose a license

### Step 4: Create Repository
Click the green **"Create repository"** button

### Step 5: Come Back Here
Once the repository is created, come back and we'll push your code!
