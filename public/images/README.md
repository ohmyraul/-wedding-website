Place your wedding images in this directory with the following filenames:

1. **hero.jpg** - Main hero image (couple photo for the landing section)
2. **alysha.jpg** - Photo of Alysha (for the "2015 - The First Time" story section)
3. **shubs.jpg** - Photo of Shubs (for the "2018 - The Reunion" story section)
4. **goa-scooter.jpg** - Goa/scooter adventure photo (for the "2018-2021 - Building a Life" section)
5. **cookie.jpg** - Photo of Cookie (for the "Real Bosses" section)
6. **bailey.jpg** - Photo of Bailey (for the "Real Bosses" section)
7. **kidena-house.jpg** - Kidena House photo (for the Family section)
8. **blu-missel.jpg** - Blu Missel venue photo (for the Celebration section)

## Image Format

- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- Recommended size: 1200px width or larger
- Optimize images for web (compressed but good quality)

## Additional Optional Images

If you have more images, you can add them with descriptive names and we can add them to additional sections!
